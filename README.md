# footprints

A lightweight React + Vite web project built with TypeScript, CSS, and HTML.

## Live At

- https://footprints-rapk.onrender.com

## Overview

`footprints` is a small web app created using React, Vite, and TypeScript. It is designed as a simple frontend project with modern tooling, focused on fast development and deployment.

## Tech Stack

- React
- TypeScript
- Vite
- CSS
- HTML

## Features

- Fast development setup with Vite
- Type-safe components with TypeScript
- Deployed live on Render
- Local storage , nothing gets out of your computer :) 
- You dont have to open console to look at your Local Storage , it shows you in real time
## Project Structure

- `src/` — application source code
- `public/` — static assets
- `index.html` — entry HTML page
- `package.json` — dependencies and scripts
- `tsconfig.json` — TypeScript configuration
- `vite.config.ts` — Vite configuration

## Run Locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.
